Matrix Inverse via Solving Linear Systems

Let A ∈ ℝn×n be nonsingular (invertible). Then the inverse exists and is unique, and A−1A = I and AA−1 = I.

Column-wise formulation

Write the inverse as a matrix of unknown columns:

A−1 = [x1 x2 ⋯ xn],    xi ∈ ℝn.

Write the identity matrix using standard basis vectors:

I = [e1 e2 ⋯ en].

Here ei is the vector with a 1 at entry i and zeros elsewhere.

Using AA−1 = I:

A[x1 x2 ⋯ xn] = [e1 e2 ⋯ en].

Matrix multiplication acts column-by-column, so this is equivalent to n independent linear systems:

Ax1 = e1
Ax2 = e2
⋮
Axn = en
Uniqueness: Because A is nonsingular, each system Axi = ei has a unique solution.

Solve efficiently using one factorization

Instead of solving each system from scratch, factorize A once and reuse the factorization for all right-hand sides.

LU factorization (with pivoting)

In practice, compute:

PA = LU,

where P is a permutation matrix, L is lower triangular, and U is upper triangular. For each i:

Axi = ei
⇔ PAxi = Pei
⇔ LUxi = Pei.

Solve in two triangular steps:

  • Forward substitution: solve Lyi = Pei
  • Backward substitution: solve Uxi = yi

QR factorization

Compute:

A = QR,

where Q is orthonormal and R is upper triangular. For each i:

Axi = ei
⇔ QRxi = ei
⇔ Rxi = QTei.

Then solve Rxi = QTei by backward substitution.

Assemble the inverse

After computing all columns, stack them to form the inverse:

A−1 = [x1 x2 ⋯ xn].
